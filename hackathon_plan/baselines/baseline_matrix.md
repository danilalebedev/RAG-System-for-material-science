# Baseline Matrix

Цель baseline-ветки - быстро получить рабочие сравнимые варианты, а не сразу строить идеальную систему.

| ID | Baseline | Что строим | Зачем нужен | Ожидаемые плюсы | Ожидаемые минусы | Метрики |
|---|---|---|---|---|---|---|
| B0 | Keyword search | BM25/TF-IDF по чанкам | Минимальная нижняя граница | Быстро, прозрачно, работает без LLM | Плохо ловит синонимы и сложные связи | Recall@k, nDCG@10, latency |
| B1 | Dense Vector RAG | Embeddings + vector index + LLM answer | Главный простой baseline | Хороший семантический поиск | Теряет структуру эксперимента | Recall@k, answer faithfulness |
| B2 | Hybrid RAG | BM25 + dense + reranker | Сильный production-like baseline | Стабильнее на числах, марках сплавов, аббревиатурах | Еще не показывает причинные связи | Recall@k, MRR, citation accuracy |
| B3 | RECIPER-style dual view | Paragraph chunks + procedure summaries | Проверка гипотезы из локального PDF | Лучше для вопросов про режимы и процедуры | Нужен этап summary/extraction | Recall@1, nDCG@10, QA quality |
| B4 | Graph-only search | Entity/relation graph + graph traversal | Проверить пользу графа отдельно | Отлично объясняет связи | Плохо отвечает без текстового контекста | graph hit rate, relation precision |
| B5 | Hybrid GraphRAG | B2/B3 + graph traversal + evidence pack | Основной кандидат на финал | Баланс точности, объяснимости и скорости | Сложнее отладка | все метрики + UX demo score |

## Рекомендуемая последовательность

1. Реализовать B0 и B1 в первые часы после получения данных.
2. Добавить B2 как надежный baseline для финального сравнения.
3. Реализовать B3 на 20-50 самых релевантных документах, если хватает времени на summary.
4. Построить B4/B5 через experiment-centric граф: не пытаться сразу делать сложную онтологию.
5. Финальный MVP презентовать как B5, но обязательно показать таблицу сравнения B0-B5.

## Минимальные контрольные вопросы для сравнения

- Что делали по сплаву X при режиме Y?
- Как режим Y повлиял на свойство Z?
- Какие эксперименты противоречат друг другу?
- Какие свойства для материала X не измерялись?
- Какие лаборатории/команды работали с этим материалом?
- Какие гипотезы можно предложить для улучшения KPI Z?
