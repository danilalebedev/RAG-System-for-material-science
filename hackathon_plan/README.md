# Hackathon Reference Materials

Актуальный рабочий план проекта находится в корне репозитория:

- [`../DEVELOPMENT_PLAN.md`](../DEVELOPMENT_PLAN.md)

Эта папка больше не является местом для конкурирующих планов разработки. Здесь
оставлены только справочные материалы, которые не должны переопределять главный
план.

## Что осталось полезным

- `task2_actual/official_task2_requirements.md` - краткая выжимка требований
  задания.
- `task2_actual/dataset_inventory.md` - заметки по составу датасета.
- `yandex_ai_studio_models.md` - выбранные модели Yandex AI Studio и ссылки на
  официальные docs.
- `routerai/routerai_integration.md` - справка по RouterAI как fallback.
- `routerai/routerai_client_examples.py` - минимальные примеры клиента RouterAI.
- `sources/source_index.md` - список работ и источников.
- `ux/*` - UX flow и картинки для будущего GUI/demo.

## Что удалено

Старые roadmap/report/baseline/workstream документы удалены из рабочей папки,
потому что они дублировали друг друга и содержали устаревшие числа парсинга.
Сводная актуальная версия перенесена в `DEVELOPMENT_PLAN.md`.
