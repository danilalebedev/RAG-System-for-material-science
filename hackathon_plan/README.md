# План подготовки решения для кейса "Научный клубок"

Эта папка содержит структурированные материалы для старта ML-хакатона Норникеля.

## Файлы

- `final_report_scientific_tangle.md` - основной отчет на 4-5 страниц: архитектура, baselines, метрики, план разработки, источники.
- `architecture/system_graph.mmd` - Mermaid-граф работы системы.
- `architecture/entity_model.mmd` - Mermaid-граф предметной модели.
- `ux/user_interaction_flow.md` - детальный сценарий взаимодействия пользователя с системой.
- `ux/user_interaction_graph_readable.png` - читаемая картинка UX-графа для отчета/слайда.
- `metadata/parsing_and_metadata_pipeline.md` - детальный план парсинга, извлечения и нормализации метаданных.
- `routerai/routerai_integration.md` - практическая инструкция по RouterAI для chat, embeddings, PDF и JSON extraction.
- `routerai/routerai_client_examples.py` - минимальные Python-примеры вызова RouterAI.
- `yandex_ai_studio_models.md` - выбранные модели Yandex AI Studio для embeddings, extraction, answer и query rewrite.
- `task2_actual/official_task2_requirements.md` - матрица требований из нового условия Task 2.
- `task2_actual/dataset_inventory.md` - инвентаризация датасета по Яндекс.Диску.
- `task2_actual/development_plan_task2.md` - обновленный структурированный план разработки под актуальное условие.
- `task2_actual/graph_retrieval_entity_design.md` - уточненная официальная модель графа, JSON extraction, vector search и graph build.
- `task2_actual/workstream_contracts.md` - разделение независимых областей разработки, границы файлов и контракты JSONL.
- `task2_actual/workstream_technical_specs.md` - техническое ТЗ по каждому workstream: требования организаторов, GUI, библиотеки, решения, критерии готовности, ссылки на работы.
- `task2_actual/task2_architecture.png` - архитектурная диаграмма обновленного решения.
- `architecture/workstream_contracts.mmd` - Mermaid-граф зависимостей между областями разработки.
- `baselines/baseline_matrix.md` - варианты baseline-решений и что в них измерять.
- `benchmarks/benchmark_plan.md` - план бенчмаркинга: retrieval, extraction, answer quality, graph quality, latency.
- `roadmap/implementation_plan.md` - пошаговый план работ до и во время хакатона.
- `sources/source_index.md` - список локальных и внешних источников, на которых основан план.
